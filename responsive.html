Responsive design

For this assignment, I implemented CSS Flex and Grid techniques to create a responsive layout for a mockup landing page. The process began by carefully examining the HTML structure in the browser, identifying the various components and layouts within the code. To arrange the elements, I utilized the power of Flex and Grid, referring to the HTML and CSS concepts covered in the course for guidance and inspiration.
